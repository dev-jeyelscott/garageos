# GarageOS

GarageOS is a mobile-first Progressive Web App for motorcycle shop management.

## Current Milestone

Milestone 0 — Project Foundation and Engineering Decisions.

## Source of Truth

Implementation must follow the approved project documentation and accepted ADRs.

## Local Development

Local setup will be expanded during Milestone 0.

### Initial verification

```bash
pnpm install
pnpm lint
pnpm typecheck
pnpm test
```

## Local Development Environment

### Requirements

- Node.js LTS
- pnpm
- Docker Desktop or Docker Engine

### Start local PostgreSQL

```bash
pnpm dev:db
```
